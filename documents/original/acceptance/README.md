# Acceptance Records

This directory stores historical manual acceptance notes.

## Contents

- `second-acceptance-verification.md`: the acceptance adjustments and checks recorded after the original MVP delivery.

## When To Read

- You need evidence of what was manually checked at that stage.
- You want to confirm whether a behavior was an acceptance-driven change rather than an architectural rule.

## Archive Note

Some references inside these files point to documents that lived in another workspace at the time. Treat those paths as historical breadcrumbs, not current instructions.
